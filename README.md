

# Test API with Node.js

This repository contains an API developed using **Node.js**, **Restify**, and **MongoDB**. It follows the **MVC-DAO pattern** to maintain clean and organized code, enabling efficient data management and interaction with a MongoDB database.

## Features

- **Architecture**: Uses MVC (Model-View-Controller) and DAO (Data Access Object) patterns for a structured, maintainable codebase.
- **Tech Stack**: Node.js, Restify, MongoDB

This project demonstrates clean code practices for API development in Node.js.
